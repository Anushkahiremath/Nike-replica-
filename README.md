# NikeReplica – Modern Sneaker E-Commerce Website

NikeReplica is a modern and responsive sneaker e-commerce website inspired by premium sportswear platforms. The project focuses on delivering a premium shopping experience with stylish UI design, smooth animations, and visually appealing sneaker showcases.

## Features

- Modern and responsive UI
- Smooth page transitions and hover animations
- Interactive sneaker showcase section
- Animated product cards
- Mobile-friendly responsive layout
- Dynamic navigation experience
- Smooth scrolling effects
- Attractive hero section with motion effects
- Optimized frontend performance
- Clean and scalable project structure

## Animations Included

- Fade-in page transitions
- Hover scale effects on products
- Animated buttons and cards
- Smooth navbar transitions
- Hero section motion effects
- Scroll-based reveal animations
- Interactive image hover effects

## Tech Stack

### Frontend
- React.js
- TypeScript / JavaScript
- Vite
- Tailwind CSS
- Framer Motion

### Backend
- Node.js
- Express.js

### Deployment
- Vercel

### Version Control
- Git & GitHub
